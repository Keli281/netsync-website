# Netsync Limited - Corporate Website

![Netsync Limited](https://netsync-ltd.vercel.app/og-image.jpg)

## 📌 Overview

This is a modern, responsive, static corporate website for **Netsync Limited** – a Kenyan company offering Security, Energy, Civil Works, Office Solutions, and General Supplies.

Built with **pure HTML5, CSS3, and vanilla JavaScript** – no frameworks, no back-end, no third-party form services.

## 🚀 Tech Stack

- HTML5 (Semantic)
- CSS3 (Custom properties, Flexbox, Grid, Animations)
- Vanilla JavaScript (Intersection Observer, smooth scrolling, form validation)
- Google Fonts (Inter)
- Font Awesome (Icons)

## 🛠️ Deployment

### Vercel (Recommended)

1. Push this code to a GitHub repository
2. Go to [Vercel](https://vercel.com) and import the repo
3. Deploy with default settings (no build commands needed)
4. Your site will be live at `https://your-project.vercel.app`

### Custom Domain (Truehost)

1. Buy your domain from Truehost
2. In Vercel, go to Settings → Domains
3. Add your domain (e.g., `netsync.co.ke`)
4. Update DNS records at Truehost:
   - Add CNAME record pointing to `cname.vercel-dns.com`
   - Or use Vercel's nameservers

## 📝 Contact Form

The form logs submissions to the **browser console** (for now). To connect to a real backend:

- Replace the `console.log` in `script.js` with a `fetch` POST request
- Or use a service like EmailJS, Formspree, or your own API
